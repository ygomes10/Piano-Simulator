🎹 Piano Simulator

Um simulador de piano virtual desenvolvido com HTML, CSS e JavaScript, permitindo que o usuário toque notas musicais diretamente pelo teclado do computador ou clicando nas teclas exibidas na interface.

O projeto foi criado com o objetivo de praticar conceitos de desenvolvimento web, manipulação do DOM e eventos em JavaScript, proporcionando uma experiência interativa e intuitiva para o usuário. Simuladores de piano em JavaScript normalmente utilizam eventos de teclado e reprodução de áudio para criar uma experiência musical em tempo real.

🚀 Funcionalidades
🎵 Reprodução de notas musicais em tempo real
⌨️ Controle através do teclado do computador
🖱️ Interação por cliques nas teclas do piano
🎨 Interface amigável
🔊 Feedback sonoro imediato ao pressionar as teclas

🛠️ Tecnologias Utilizadas
HTML5
CSS3
JavaScript (ES6+)
🎯 Objetivos do Projeto
Praticar manipulação de eventos com JavaScript
Trabalhar com reprodução de áudio na web
Aprimorar habilidades de desenvolvimento Front-End
Desenvolver interfaces interativas
